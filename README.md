# Class Forum 🚀📚

## Overview

**Class Forum** is a **full-stack discussion platform** built to help students collaborate, ask questions, and share knowledge related to their coursework.
The platform functions similarly to a lightweight academic forum where students can create posts, participate in discussions, and interact with content through voting.

The project focuses on building a **scalable discussion system with an intuitive user interface**, combining frontend design, backend logic, and database integration.

The application is **live and deployed**, allowing users to interact with the platform in real time.

---

## Key Features

* 📝 Create and view discussion posts
* 💬 Comment on discussions and participate in conversations
* 👍 **Upvote / Downvote system** to highlight useful answers
* 👤 User authentication (login/signup)
* 🎨 **Clean and responsive UI** for smooth user interaction
* 🗂 Organized forum-style discussion threads
* 🌐 **Live deployed web application**
* ⚡ Full-stack architecture integrating frontend, backend, and database

## Architecture

The application follows a **full-stack architecture**:

Frontend
Handles user interface, forms, and interactive elements.

Backend
Processes user requests, manages authentication, and handles business logic such as posting, commenting, and voting.

Database
Stores users, posts, comments, and vote counts.
